# network-security

pfSense firewall (Netgate 2100) running the four-VLAN architecture for the home: VLAN 1 management/personal, VLAN 10 cameras (no internet), VLAN 20 IoT, VLAN 30 guest WiFi. This section will hold pfSense config backups, firewall rule documentation, VLAN reference, and eventually a local dashboard app that surfaces firewall health, VLAN traffic stats, and alert events pulled from the pfSense API.

**Status:** Not yet started.
