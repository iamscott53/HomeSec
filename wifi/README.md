# wifi

Covers both the main household WiFi security (SSID hardening, WPA3 configuration, AP firmware tracking, rogue-AP detection, client inventory on VLAN 1 / VLAN 20) and the VLAN 30 guest WiFi (isolated SSID, internet-only access, no LAN reachability, bandwidth limits, and eventually a captive portal). Any future app in this section will surface connected-client lists, signal health, firmware status, and anomalous guest-network behavior.

**Status:** Not yet started.
