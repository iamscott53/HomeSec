# switch

Ubiquiti USW-Lite-16-PoE (16-port managed, 8x PoE, 45W budget) — the main LAN switch in the office rack. This section will hold the port map (which run/device lands on which port), PoE budget accounting for the G4 Doorbell Pro and any other PoE loads, VLAN tagging reference per port, and eventually a small monitoring app that watches link state, port errors, and PoE power draw.

**Status:** Not yet started.
